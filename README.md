安装 Installation
-----------------

推荐使用 [composer](http://getcomposer.org/download/) 安装。

通过执行命令

```
php composer.phar require --prefer-dist ipaya/weibo-sdk
```

或添加

```
"ipaya/weibo-sdk": "*"
```

到 `composer.json` 文件的 `require` 段。
